git checkout <ключи> <имя ветки> 
    Переключает пользователя на другую ветку
    -b Если вы передадите этот ключ, то команда создаст новую ветку, а затем переключится на нее.
git branch
    <имя ветки>
    Создает новую ветку.
    <ключи>
        По умолчанию выводит список локальных веток. С ключами -r, -a можно вывести, соответственно,
        либо только удаленные ветки, либо все ветки.
        -r От англ. remote – удаленный. С этим ключом будут выведены только удаленные ветки. О них мы поговорим в следующем уроке. 
        -a От англ. all - все. С этим ключом будут выведены и локальные, и удаленные ветки

        git push -u origin master
        git pull origin master --allow-unrelated-histories
        git push origin --delete <branchName>
